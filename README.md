# Readme

## Setup

Run `script/setup` and ensure that you have the `SLACK_API_TOKEN` configured in `.env`.

## Run

Run bot with `script/server`.

## Commands

### Command list

    > user [8:37 PM]: @bot: commands

    > bot [8:37 PM]:
    > commands
    > @bot: jokeme # tells you a joke
    > @bot: keyword # ruby ri doc explaination

### Jokeme
